# Tender Beginnings Wellness

A proof-of-concept website for a doula and maternal wellness business offering:

- **Certified Doula Services** — Birth support and advocacy
- **Birth Education Classes** — Via Zoom
- **Lactation Counseling** — Breastfeeding support
- **3D Ultrasound Imaging** — Bonding before birth
- **Postpartum Support** — Including PPD resources

## Tech Stack

- Next.js 16 (App Router)
- MUI (Material UI) v7
- TypeScript

## Getting Started

```bash
pnpm install
pnpm dev
```
