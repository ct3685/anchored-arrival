# Trevor / Ranch Squad Redesign Spec

## Project goal
Redesign the Trevor / Ranch Squad site so it no longer feels like a reskinned clone of Agent Morgie. The new site should feel like its own brand world: a high-energy, over-the-top, country-meets-fight-night experience built for a TikTok Live creator.

This should feel like:
- a rodeo poster
- a WWE entrance package
- a dive-bar neon sign
- a western merch drop
- a live-event landing page

This should **not** feel like:
- a generic influencer link-in-bio site
- a soft, polished creator portfolio
- a copy of Agent Morgie with different colors
- a rustic farmhouse aesthetic
- a cowboy costume theme

---

## Core brand concept
**Brand name:** Trevor / Ranch Squad  
**Brand position:** Trevor is not just a creator. He is the ringleader of a live internet rodeo.  
**Primary tone:** loud, dramatic, chaotic, rallying, playful, confident  
**Secondary tone:** gritty, sharp, unfiltered, western, showman energy

### Brand phrases to use intentionally
Use these sparingly but prominently:
- Com'On
- Ranch Squad
- Gooder than shit
- No Power Ups
- Pull up
- Suit up
- The ranch is live
- Real ones only
- Tonight at the ranch
- Raise hell

Do not overuse all phrases on every section. Give each phrase a place so the site feels designed, not stuffed.

---

## Chosen creative direction
## Neon Rodeo Panic
This is the main direction for the redesign.

### Design thesis
The site should feel like a western event flyer for a creator who goes live like he's walking into an arena. The experience should feel kinetic and theatrical, but still clean enough to be usable.

### Aesthetic references
- county fair lights at night
- rodeo arena signage
- distressed metal and belt buckles
- western turquoise jewelry accents
- ticket stubs and posters
- red neon and amber floodlights
- Oklahoma bar flyer energy

---

## Visual system

### Color palette
Use a tighter, more intentional palette.

**Base colors**
- Smoke Black: `#0D0A09`
- Coal Brown: `#18110E`
- Dark Leather: `#241813`

**Primary accents**
- Electric Amber: `#F5A623`
- Rodeo Red: `#D33A2C`
- Turquoise Stone: `#33C6C0`

**Support tones**
- Bone: `#F2E6D8`
- Brass: `#B88746`
- Dust: `#8B7463`

### Color usage rules
- Use black, coal, and leather for the main background layers
- Use amber for important highlights and active states
- Use rodeo red for impact moments, warnings, live states, and dramatic gradients
- Use turquoise as a signature accent, never as the main dominant color
- Use bone for readable text and contrast points
- Avoid rainbow gradients or too many glowing colors at once

---

## Typography direction
Use contrast between an aggressive display face and a clean utility face.

### Heading style
Use a condensed western or slab-inspired display font for:
- hero headlines
- big section titles
- CTA headings
- standout phrases like COM'ON

The display font should feel like:
- fight poster
- western signboard
- event flyer

### Body/UI font
Use a clean sans serif for:
- paragraphs
- navigation
- buttons
- metadata
- supporting copy

### Typography rules
- Big headlines should be bold and compact
- Small labels should feel like stamped utility text
- Use occasional all-caps, but not for every paragraph
- Avoid script fonts, rustic cursive, or novelty cowboy fonts

---

## Shape language
This is one of the most important parts of making the site feel original.

### Replace circles and hearts with western structural motifs
Preferred motifs:
- horseshoe outlines
- belt buckle frames
- ticket stub corners
- clipped metal plate corners
- cattle tag silhouettes
- sheriff-star bursts used sparingly

### Rules for shapes
- Do not use plain circular profile and image holders
- Do not use heart particles
- Do not turn every element into a literal cowboy hat shape
- Cowboy hats are best as a small accent icon or particle, not the main card structure
- Choose 2–3 structural motifs and repeat them consistently

### Best shape choices for implementation
Use these across the site:
1. **Horseshoe outline** for particles, dividers, tiny icon moments
2. **Buckle plate frame** for hero portrait or featured image
3. **Ticket-stub button/card edges** for CTAs and link modules
4. **Cattle-tag gallery tiles** for image collections

---

## Texture and effects

### Good effects
- subtle dust grain overlay
- distressed chrome or brushed brass highlights
- soft light bloom around amber or red accents
- shallow motion blur on entering elements
- low-opacity vignette

### Avoid
- sparkly magical particles
- soft pastel glows
- generic glassmorphism everywhere
- overly blurred neon blobs with no purpose
- floating hearts

---

## Motion direction
Motion should feel like signage, hype, and stage presence.

### Motion ideas
- subtle flicker on LIVE labels
- slow crawl or ticker text for slogans
- cards reveal upward like poster cards being slapped onto a wall
- background lights pulse slightly, not constantly
- hover states should feel snappy, not floaty
- image cards can jolt or brighten slightly on hover

### Motion do-not-do
- do not make everything bounce
- do not make everything wobble
- do not use cute motion language
- do not make animations feel playful in a childish way

---

## Component design rules

### Buttons
Buttons should feel like event tickets or gear labels.

**Button treatment ideas**
- ticket stub side notches
- subtle metallic border
- bold uppercase label
- small sublabel under main action if needed

**Examples**
- Pull Up Live
- Suit Up
- Join the Ranch Squad
- Watch the Chaos
- Shop the Gear

### Cards
Cards should not look like generic SaaS cards.

Use:
- clipped corners
- buckle-inspired border treatment
- layered plate effect
- edge highlights in amber or brass

### Badges and stickers
Use sparingly for impact:
- LIVE NOW
- REAL ONES ONLY
- NO POWER UPS
- COM'ON

These can appear like little starburst stickers, venue stamps, or corner tags.

## Technical and implementation notes for the other AI service

### General instruction
Redesign existing pages without deleting site structure unless necessary. Prioritize component and visual transformation over rebuilding everything from scratch.

### CSS/implementation priorities
- use reusable tokens for color and spacing
- implement a small set of branded motifs consistently
- use `clip-path`, pseudo-elements, masks, or layered borders for custom shapes
- keep responsive behavior simple and durable
- ensure custom shapes degrade gracefully on small screens

### Shape implementation guidance
Good candidates for CSS build:
- ticket notches using pseudo-elements
- clipped corners using `clip-path`
- cattle-tag style blocks using polygon clip-paths
- buckle frame using layered bordered containers
- horseshoe outlines as SVG icons or pseudo-element decorations

### Do not overcomplicate with too many one-off shapes
Build a system, not a gimmick.

## Success criteria
The redesign is successful if:
- it no longer feels like an Agent Morgie clone
- it no longer feels like a standard creator microsite
- the site has a repeatable western visual language
- Trevor's brand feels louder, more specific, and more ownable
- the shapes, typography, and CTAs all feel like the same world
- the homepage feels like an event, not a biography

## Failure criteria
The redesign fails if:
- it just swaps colors and icons
- cowboy hats are used as a gimmick everywhere
- circles and hearts are mostly still intact
- the site still reads like a generic creator template
- the merch section still feels like a recycled CTA
- the design feels rustic-cute instead of dramatic and commanding
